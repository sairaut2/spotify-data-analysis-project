## 📌 Project Title
Spotify Songs Analysis: Understanding Music Trends and Popularity

---

## 📖 Problem Statement
This project analyzes the Spotify Features dataset to understand patterns in music such as popularity, genre, and audio characteristics. The goal is to identify key factors that influence song popularity and trends in the music industry.

---

## 📂 Dataset Information
- Source: Kaggle
- Dataset Name: Spotify Features Dataset

### Features:
- popularity (numeric)
- genre (categorical)
- artist_name
- track_name
- tempo
- loudness
- duration_ms

---

## 🛠️ Tools & Technologies
- Python
- Pandas
- NumPy
- Matplotlib
- Seaborn
- Scikit-learn

---

## 📊 Analysis Performed
- Data Cleaning
- Statistical Analysis
- Data Visualization (Histogram, Bar Chart, Boxplot, Heatmap)
- Machine Learning Model (Linear Regression)

---

## 📈 Key Insights
1. Most songs have medium popularity.
2. Certain genres dominate the dataset.
3. Some songs are highly popular (outliers).
4. Tempo does not strongly affect popularity.

---

## 💡 Recommendations
1. Focus on trending genres to increase song popularity.
2. Consider multiple audio features for better prediction of success.
